Folder Structure


Name your app "test_app". The whole app directory structure should look like below: 

	            	
/test_app
    - app.py
    - /static
        - style.css
    - /templates
        - base_layout.html
        - index.html
        - about_test_app.html

Templates

Base Layout


Nothing fancy. Your base html layout should be similar below. You can do your own styling.

	        		
<!doctype html>
<html>
<head>
<link type="text/css" rel="stylesheet" 
	href="{{ url_for('static', filename='style.css') }}" />
</head>
<body>
    {% block content %}
    	
Hello world!

    {% endblock %}
</body>
</html>

Extending Base Layout


	        		
{% extends "base_layout.html" %}

{% block content %}
	Your HTML code here...
{% endblock %}

Running your App

To invoke and start your first web application, in the command line, type:

		    	
$ python app.py

The welcome page (default page) should display the name of your team. ie. navigating to
