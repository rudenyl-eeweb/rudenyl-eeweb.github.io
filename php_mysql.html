PHP: Introduction

Variable Declarations

A PHP variable can have a short name (like x and y) or a more descriptive name (color, first_name, total_volume).

Rules for PHP variables:

  • a variable starts with the dollar $ sign, followed by the name of the variable
  • a variable name must start with a letter or the underscore character
  • a variable name cannot start with a number
  • a variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )
  • variable names are case-sensitive so $color and $COLOR are two different variables

Arrays

An array in PHP is actually an ordered map, which associates values to keys. It takes any number of comma-separated key => value pairs as arguments. An array can be declared using the array() language construct, or [ ] (as of PHP5.4).

Array keys can either be an integer or a string. The values can be of any type.

Arrays (cont)

Additionally the following key casts will occur:

  • Strings containing valid integers will be cast to the integer type. E.g. the key "8" will actually be stored under 8. On the other hand "08" will not be cast, as it isn't a valid decimal integer.
  • Floats are also cast to integers, which means that the fractional part will be truncated. E.g. the key 8.7 will actually be stored under 8.
  • Bools are cast to integers, too, i.e. the key true will actually be stored under 1 and the key false under 0.
  • Null will be cast to the empty string, i.e. the key null will actually be stored under "".
  • Arrays and objects can not be used as keys. Doing so will result in a warning: Illegal offset type.

Accessing Arrays

					
<?php
$tmp = array(		# offset checking
    1 => 'apple',	// index: 1
    "two" => 2,		// index: ? -- not counted
    array(1, 2)		// index: 2 -- since previous index is not counted
);

// get array count
$count = count($tmp);	# $count --> 3

echo $tmp[0], "\n";	# error --> Undefined offset: 0
echo $tmp[1], "\n";	# apple
echo $tmp['two'];	# 2

echo $tmp[two];		# will display '2' but will throw a notification
			# Notice: Use of undefined constant two - assumed 'two'

echo $tmp[2];		# error --> Array to string conversion
			# use var_dump() or print_r() instead
			# since the value is an array

echo $tmp[2][1];	# 2

$n = count($tmp[2]);	# array length --> 2
echo $tmp[2][$n - 1];	# 2
?>

Variables Scope


The scope of a variable is the part of the script where the variable can be referenced/used.

PHP has three different variable scopes:

  • local
  • global
  • static

Variables Scope

		        		
<?php
$offset = 1; # global scope

function global_var_test($x)
{
    return $x + $y + $GLOBALS['offset'];
}

function global_var_test_2($x)
{
    global $offset;

    return $x + $y + $offset;
}

function local_var_test($x)
{
    $offset = 1; # local scope

    return $x + $offset;
}

echo global_var_test(3);	# 4
echo global_var_test_2(3);	# 4
echo local_var_test(3);		# 4
?>
		        		
		        	
		        		
<?php
# normally, when function is completed,
# local variables are deleted
# but with static scope, the variable
# retains the value

# static scope
function static_var_test()
{
    static $offset = 0;

    return $offset++;
}

echo static_var_test();		# 0
echo static_var_test();		# 1
echo static_var_test();		# 2
echo static_var_test();		# 3
echo static_var_test();		# 4
?>

What is a class?

A class is a collection of variables and functions working with these variables. Class variables are defined by the var keyword or any of the visibility keywords (public, private, protected); and functions/methods are declared by the keyword function.

Example of Class

					
<?php
/**
 * Class structure
 */
class TestClass
{
    var $name = 'MyClass';
    protected static $type = 'class';

    function __construct( $name='' )
    {
        if (!empty($name)) {
            $this->name = $name;
        }
    }

    public function className()
    {
        return $this->name;
    }

    public static function getType()
    {
    	return self::$type;
    }
}
?>

Property Visibility

Class properties must be defined as public, private, or protected. If declared using var, the property will be defined as public. 

					
<?php
class TestClass
{
    var $name = 1;                  # public
    public $context = 'public';     # public
    protected $data = 'protected';  # protected
    private $secret = 'secret';     # private

    function showData()
    {
        echo $this->name,
             $this->context,
             $this->data,
             $this->secret;
    }
}


$o = new TestClass;
echo $o->name;      // returns 1
echo $o->context;   // returns 'public'
echo $o->data;      // Fatal error: Cannot access protected property
echo $o->secret;    // Fatal error: Cannot access private property

$o->showData();     // displays 1, 'public', 'protected' and 'secret'
?>

Method Visibility

Class methods may be defined as public, private, or protected. Methods declared without any explicit visibility keyword are defined as public. 

					
<?php
class TestClass
{
    function publicFn1() { return 'public enemy #1'; }
    public function publicFn2() { return 'public enemy #2'; }
    protected function protectedFn() { return 'protected by the police or politicians'; }
    private function privateFn() { return 'under-the-table'; }

    public static function showAll()
    {
        echo self::publicFn1(),
             self::publicFn2(),
             self::protectedFn();
             self::privateFn();
    }
}


$o = new TestClass;
echo $o->publicFn1();    // returns 'public enemy #1'
echo $o->publicFn2();    // returns 'public enemy #2'
echo $o->protectedFn();  // Fatal error: Call to protected method
echo $o->privateFn();    // Fatal error: Call to private method

TestClass::showAll();    // displays all :)
?>

Method Chaining

Setting up mutator/setter methods to return the calling or original object, which in turn can call other methods 

					
<?php
class TestClass
{
    public $name = [];

    public function setFirstName($first_name) {
        $this->name[] = $first_name;
        return $this;
    }

    public function setLastName($last_name) {
        $this->name[] = $last_name;
        return $this;
    }

    public function display() {
        return implode(' ', $this->name);
    }
}


$o = new TestClass;
echo $o->setFirstName('John')->setLastName('Doe')->display();  // returns 'John Doe'
?>

Object Inheritance

Inheritance is a well-established programming principle, and PHP makes use of this principle in its object model. This principle will affect the way many classes and objects relate to one another.

For example, when you extend a class, the subclass inherits all of the public and protected methods from the parent class. Unless a class overrides those methods, they will retain their original functionality.

This is useful for defining and abstracting functionality, and permits the implementation of additional functionality in similar objects without the need to reimplement all of the shared functionality.

Object Inheritance

Example of Class inheritance 

					
<?php
class BaseClass
{
    protected $class = 'baseclass';
    private $secret = 'secret';

    public function getClass() {
        return $this->class;
    }

    protected function getSecret() {
        return $this->secret;
    }
}

class FirstClass extends BaseClass
{
    public $class = 'firstclass';   # property visibility is overridden

    public function getClass() {
        return get_class($this);    # method functionality is overridden
    }
}

echo (new FirstClass)->getClass();  // return 'FirstClass'
?>

How-to

The most common way to connect to a MySQL database is by using mysqli or PDO classes.

How-to (cont)

					
<?php
#
# mysqli
#
$dbh = new mysqli('localhost', 'webuser', 'webpass', 'your_db');
$result = $dbh->query("SELECT `your_column_name` FROM `your_table`");
$row = $result->fetch_assoc();
echo $row['your_column_name'];

$dbh->close(); # close database connection


#
# PDO
#
$dbh = new PDO('mysql:host=localhost;dbname=your_db', 'webuser', 'webpass');
$result = $dbh->query("SELECT `your_column_name` FROM `your_table`");
$row = $result->fetch(PDO::FETCH_ASSOC);
echo $row['your_column_name'];

$dbh = null; # close database connection
