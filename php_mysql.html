PHP: Introduction

PHP: Hypertext Preprocessor (PHP) is a server-side scripting language designed for web development but also used as a general-purpose programming language.

  • started by Rasmus Lerdof which originally stands for "Personal Home Page"
  • does not need to be compiled before it is used
  • widely ported and can be deployed on most web servers on almost every operating system and platform, free of charge.

Releases

PHP has evolved dramatically over the years and supports a variety of programming techniques.

Code Syntax

Start of Code

Every piece of PHP code begins with <?php (or the abbreviated <? if your server is configured to handle that).

End of Code

The way to signify that the PHP code is finished is by adding ?> at the end but not required.

Code block

Every code block, with a few exceptions, each separate instruction that you write will end with a semicolon.

Naming Variables

PHP variables start with the dollar sign $. You can name your variable anything you like so long as it follows these rules:

  • it starts with a letter
  • it is made up of letters, numbers, and the underscore ( _ ) character, ie $Test1_2_3
  • case-sensitive, so $Test and $test are not the same

Compound Words

For code readability and sanity, we can apply these compound words naming styles.

Style Context Variable Name Function Name
camelCase Hello world $helloWorld function helloWorld() {...}
PascalCase Hello world $HelloWorld function HelloWorld() {...}
snake_case Hello world $hello_world function hello_world() {...}

Conditional Logic

	        		
<?php
if (true) echo 'always executed';
if (false) echo 'not coming here ever';

$a = 'false';
if ($a) echo 'yo!';
if ((int)$a) echo 'yay!';

$b = null;
if ($b) echo 'null wont be here';
else if ($b == "null") echo 'not here either';
else if ($b === null) echo 'null is here';
else {
	echo "hmmm...";
}
?>

Loops

	        		
<?php
$n = array(1,2,5,10);

foreach ($n as $whatever) {
	echo $whatever .'-';
}

# this will throw an exception
# because $n is an index-based array
for ($i=0; $i<5; $i++) {
	echo $n[$i];
}
?>

Loops include C-like implementations, ie do..while and while loops.

Switch

An example of a switch statement 

	        		
<?php
$chicken_is = 'sick';

switch ($chicken_is) {
    case 'sick':
        echo "The chicken is {$chicken_is}. Poor little chicken.";
        break;

    case 'dead':
    	echo 'You killed the chicken! I heychu!';
    	break;

    default:
    	echo 'The chicken looks okaaay --- so far';
    	break;
}
?>
